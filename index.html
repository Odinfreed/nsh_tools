📖 使用说明

1. 数据导入： 可通过“导入JSON文件”或直接粘贴JSON数据。建议使用以下提示词让豆包生成JSON格式内功数据：

📌 单批识别提示词（发给豆包，用于识别单个内功）
我将给你发一些图片，请把它按照Json格式去生成文字。主要内容要求如下：
内功名称
蕴含元素（例如图中是 4 金 1 火）
属性加成
（特别注意，五韵谣不用统计蕴含元素）
具体格式如下（示例）：
{
  "neigong_list": [
    {
      "name": "灼锋净域",
      "elements": "1木4火",
      "attributes": {
        "气血上限": "+47834",
        "破盾": "+945",
        "破防": "+1500",
        "身法": "+213"
      }
    },
    {
      "name": "五韵谣",
      "elements": null,
      "attributes": {
        "气血上限": "+47834",
        "命中": "+429"
      }
    }
  ]
}
🔗 汇总提示词（所有内功识别完毕后发送）
综上所述 将所有内功进行统计，确保所有统计的是同一个Json文件

2. 手动勾选组合： 在排行榜中勾选内功（最多6个，同名不可重复），点击“计算勾选组合收益”即可查看总收益及元素校验。

3. 自动推荐搭配： 点击“开始评分”后，系统会自动计算所有内功分数，并给出尽可能满足三火三金一木（火≥12、金≥12、木≥4）且稀有内功≤3的最高总分组合。

4. 手机适配： 页面已适配手机，可横屏查看表格，所有按钮均支持触屏。
